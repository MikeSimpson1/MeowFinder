WINDOWS 10 ONLY

PRE-REQUISITES:
Chromedriver: https://www.youtube.com/watch?v=dz59GsdvUF8
Python3: https://www.youtube.com/watch?v=V_ACbv4329E
Webdriver: https://www.youtube.com/watch?v=gVXcVcTRXd0

HOW TO USE:
1. Extract zip to desired location
2. Open MeowFinder.py in any python editor
3. Edit lines 5,6,7,8 with desired information (line 5 crucial)
4. Save
5. Open CMD, navigate to folder with MeowFinder.py
6. Run command: MeowFinder.py

My initial use was to make this to find new kitten postings in my area, hence the name MeowFinder, but can be used with any Kijiji page.
